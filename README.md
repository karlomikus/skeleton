# Slim Skeleton
Quick start application based on Slim Framework 3 and open source components.

![BEST SKELETON](https://hydra-media.cursecdn.com/dota2.gamepedia.com/f/fc/Skeleton_King.png)

## Features
* Included basic providers
* Session handling
* Authentication
* Unit tests

## Installation
1. Git clone
2. Create database
3. Setup .env
4. Start developing
